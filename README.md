# Computer-Security
Implementation of the below techinques for two files: smallfile.txt and testdata.txt
<ul>
  <li>aes_with_CTR.py: Implementation of AES in CTR mode using 128-bit and 256-bit keys</li>
  <li>dsa_cryptography.py: Sign two files using 2048-bit and 3072-bit DSA key</li>
  <li>hash.py: Compute a hash of each of the files using hash functions SHA-256, SHA-512, and SHA3-256</li>
  <li>rsa.py: Create a 2048-bit and 3072-bit RSA key, encrypt and decrypt the files above with PKCS #1 v2 padding</li>
</ul>
